Gestor De Ferias

Proyecto realizado en pareja para el desarrollo de una plataforma web orientada a la gestión de eventos y ferias, adaptable también a museos u otras instituciones culturales. El objetivo es facilitar la organización, programación y administración de actividades de forma clara y eficiente.

Detalles Técnicos:

Para este proyecto aplicamos los mismos conocimientos técnicos utilizados en BeatBite: desarrollo con Laravel, uso de PHP, Eloquent para la gestión de bases de datos, Vue.js para el frontend dinámico, e integración de APIs. Trabajamos bajo el modelo de controladores, modelos y vistas, aprovechando la arquitectura de Laravel para construir una aplicación robusta y escalable.
